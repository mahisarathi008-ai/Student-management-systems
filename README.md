Student Management System
Overview

The Student Management System is a console-based application developed in C programming. It helps users manage student records efficiently through a simple menu-driven interface. The project demonstrates the use of structures, arrays, loops, conditional statements, and functions in C.

This project is suitable for beginners learning C programming and basic data management concepts.

Features
Add new student records
Display all student details
Search students by Roll Number
Calculate average marks
Menu-driven user interface
Simple and easy-to-understand code
Beginner-friendly project
Technologies Used
C Programming Language
GCC Compiler
Command Line Interface (CLI)

How It Works
Run the program.
Select an option from the menu.
Enter student details when adding records.
View all stored records.
Search for a student using Roll Number.
Calculate average marks of all students.
Exit the application.
Compilation

Using GCC Compiler:

gcc student_management_system.c -o student
Execution

Windows:

student.exe

Linux / macOS:

./student
